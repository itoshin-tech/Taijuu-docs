## ストア掲載情報

### アプリタイトル（30文字以内）
```
DietGraph - グラフにこだわった体重管理アプリ
```

### 簡単な説明（80文字以内）
```
日々の体重変化を使いやすいグラフとカレンダーで確認できます。
イベント、メモ、体重の上限と目標の設定も記録して、体重変化との関係を把握できます。
```

### 詳しい説明（4000文字以内）
```
グラフの使いやすさ、カレンダーの見やすさにこだわった体重管理アプリです。
体重だけでなくイベントも記録してグラフに重ねて表示ができます。
設定した上限体重と目標体重でカレンダーのセルを色分けして表示されます。
毎日のメモも記録でき、カレンダーに表示されます。

【主な機能】
・ドラムロール式の入力で、素早く体重を記録。
・食事制限成功、運動、お通じなどのイベントを簡単記録。
・折れ線グラフにイベントを重ねて表示するので、体重が下がった原因が一目でわかる！
・上限体重、目標体重ラインをグラフ上に表示、ドラッグで簡単に変更可能。
・カレンダーには各日々のイベントとメモを表示。
・カレンダーのセルの色は、上限と目標の範囲で色分けするので、体重が少くなった日が一目でわかる！
・データの外部保存・読み込みが可能。
```
